# Intro-to-AI---Team-3

## 프로젝트명
수업자료 기반 검색형 AI 튜터

## 프로젝트 소개
PDF 또는 텍스트 수업자료를 업로드하면 사용자의 질문과 관련된 내용을 검색하여 보여주는 학습 도우미입니다.

## 주요 기능
- 수업자료 업로드
- 텍스트 추출
- 질문 입력
- 관련 문단 검색
- 유사도 점수 출력
- 키워드 추출
- 간단한 퀴즈 생성

## 사용 기술
- Python
- Streamlit
- scikit-learn
- TF-IDF
- Cosine Similarity

## 팀원 역할
- 최종원: 검색 알고리즘 및 프로젝트 정리
- 팀원1: PDF/TXT 텍스트 추출
- 팀원2: Streamlit UI 및 결과 출력
